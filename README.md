# yt-dlp-wrapper
Simple wrapper for yt-dlp with preset options suitable for youtube, and a majority of sites or m3u8. 
Able to grab best format, optimal format, and mp3. 

## Usage
Put this in some folder, really anywhere. It'll just make a "output" folder.

## Requirements 
Requires yt-dlp. https://github.com/yt-dlp/yt-dlp
Requires ffmpeg in PATH. https://www.ffmpeg.org/
